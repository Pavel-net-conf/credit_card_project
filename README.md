# Сервис прогнозирования дефолта по кредитным картам с контейнеризацией и A/B-тестированием

## 📌 Описание проекта
**Цель проекта:** Разработка и подготовка к интеграции в банковскую production-like-среду веб-сервиса машинного обучения для прогнозирования дефолта по кредитным картам. Проект охватывает полный цикл: от разведочного анализа данных и обучения моделей до создания масштабируемого веб-интерфейса (API) с поддержкой одновременного версионирования для A/B-тестирования.

- **Домен:** Финансы / Кредитный скоринг.
- **Датасет:** Default of Credit Card Clients Dataset (UCI Machine Learning Repository).
- **Целевая переменная:** `default.payment.next.month` (1 — дефолт в следующем месяце, 0 — своевременная оплата).
